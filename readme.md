
# HashKraken
## By: David Corvaglia
### A password cracker with a GUI written in Java.
## Supported Alogrithms:
- SHA1
- MD5
- SHA256
## Features:
- Mutithreading
- GUI and CLI
- Rainbow Tables (Creation and Usage)
- Written in Java
- Top 14 Million Passwords Built In
- Ability to Add Custom Wordlists
- Brute Force
- Random Text Mode
- Auto-Caches Previous Runs
